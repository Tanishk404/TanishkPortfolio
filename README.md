# Tanishk's PS5-Themed Portfolio

A stunning, modern portfolio website with PlayStation 5-inspired design, featuring smooth animations, glassmorphism effects, and a futuristic UI.

## Features

- 🎮 **PS5-Style Loading Screen** - Authentic boot animation with your name
- 🎨 **Clean & Futuristic Design** - White/blue/black color palette
- 🎯 **Smooth Page Transitions** - Sliding animations between routes
- 📱 **Fully Responsive** - Works on all devices
- ⚡ **Fast & Optimized** - Built with React and Framer Motion

## Pages

- **Home** - Welcome page with quick stats
- **About** - Profile, skills, and experience
- **Projects** - Showcase of your work
- **Contact** - Contact form and information

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Build for production:
```bash
npm run build
```

## Technologies Used

- React 19
- React Router DOM
- Framer Motion
- Lucide React Icons
- Custom CSS with PS5 Theme

## Customization

Update your information in:
- `src/components/Home.jsx` - Home page content
- `src/components/About.jsx` - About section and skills
- `src/components/Projects.jsx` - Your projects
- `src/components/Contact.jsx` - Contact information
